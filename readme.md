## Setup
To run this project, install it locally using npm, composer, php artisan:

```
$ cd ../shokuren
$ composer install
$ npm install
$ npm run prod
$ php artisan migrate --seed
$ php artisan config:clear
$ php artisan route:cache
$ php artisan view:clear
```
